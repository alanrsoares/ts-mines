## Contributing

Clone the repository:

```bash
git clone git@github.com:alanrsoares/ts-mines.git
```

Install dependencies:

```bash
yarn install
```

Or with npm:

```bash
npm install
```

Run it:

```
yarn start
```

Or with npm:

```bash
npm start
```
